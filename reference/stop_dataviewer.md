# Stop a Specific or Most Recent Dataviewer Process

Stop a Specific or Most Recent Dataviewer Process

## Usage

``` r
stop_dataviewer(id = NULL)
```

## Arguments

- id:

  Character string specifying the process ID to stop. If NULL, stops the
  most recent background dataviewer.
